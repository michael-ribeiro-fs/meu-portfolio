# Michael Ribeiro - Portfólio

Site pessoal (portfólio) construído com **HTML, CSS e JavaScript puros** (vanilla), sem frameworks nem etapa de build. Apresenta seções de Sobre, Stack, Projetos e Contato, com conteúdo dinâmico renderizado via JavaScript e animações de revelação ao rolar a página.

🔗 Repositório: https://github.com/michael-ribeiro-fs/meu-portfolio  
🔗 Demo: https://michael-ribeiro-fs.github.io/meu-portfolio/

## ✨ Funcionalidades

- **Navegação responsiva** com menu mobile (hambúrguer) e _scroll spy_ (destaca o link da seção visível)
- **Renderização dinâmica** das seções Stack, Projetos e Contato a partir de uma fonte de dados única (`components/projectsData.js`)
- **Animações de scroll reveal** usando `IntersectionObserver`, com fallback para navegadores sem suporte
- **Ícones SVG inline**, sem dependências externas de bibliotecas de ícones
- Layout **mobile-first** com breakpoints dedicados

## 🛠️ Stack

HTML5 · CSS3 (Custom Properties, Grid, Flexbox) · JavaScript (ES Modules, classes) · Google Fonts (Inter, Space Grotesk) via arquivos locais `.woff2`
